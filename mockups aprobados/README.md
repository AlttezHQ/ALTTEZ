## Protocolo

Antes de codificar cualquier componente o sección visual, revisar los archivos aquí.

## Cómo agregar un mockup

- Nombre de archivo descriptivo: `hero-landing-v1.png`, `dashboard-home-v2.png`, etc.
- Incluir versión en el nombre si hay iteraciones
- No eliminar versiones anteriores — mover a `_archivo/` si quedan obsoletas
